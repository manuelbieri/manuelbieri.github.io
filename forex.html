My current project - ForEx

ForEx plots exchange rates and datasets in a easy and user friendly way. The data points are retrieved from the ECB and the Worldbank through there official APIs.

The desktop application is pure Python and the GUI in PyQt the official Python binding for the Qt framework.

Beside HTML/CSS is Bootstrap the main ingredient for the web application due to its great use for responsive webpages.

You can find further information and the documentation of the project on the official page.
